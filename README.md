Please see the rendered notebooks at <https://abhro.github.io/misc-pluto-notebooks>.

## Running notebooks locally

If you would like to run the notebooks on your own computer, you will need some software available, such as Julia and Pluto.jl.

1. Install Julia, following the instructions at <https://julialang.org/downloads>.

2. Install Pluto.jl, following the instructions at <https://plutojl.org/#install>.

3. Run Pluto.jl from the root of the repository with

   ```julia
   import Pluto
   Pluto.run()
   ```
